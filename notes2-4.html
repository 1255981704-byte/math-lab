01 Peeling the Onion METAPHOR
The Chain Rule is used to differentiate composite functions—functions nested inside other functions. Think of a composite function like an onion. To get to the center, you must peel it layer by layer, starting from the outside and working your way in [11]. 链式法则用于对复合函数（即层层嵌套的函数）进行求导。复合函数就像一颗洋葱：要想触及核心，你必须由外向内，一层一层地剥开它 [11]。
\[\text{Form 1: Lagrange Notation (Lagrange记法) } \quad \frac{d}{dx}[f(g(x))] = f'(g(x)) \cdot g'(x) \quad [11]\] \[\text{Form 2: Leibniz Notation (Leibniz记法) } \quad \frac{dy}{dx} = \frac{dy}{du} \cdot \frac{du}{dx} \quad [11]\]
Rule Protocol: Evaluate Inside Intact First 操作规范：内层首先保持不变
  • >> First evaluate the derivative of the outside layer, keeping the inside evaluated exactly as it is (\(f'(g(x))\)) [11].
  • >> 然后乘上内层函数的导数（\(\cdot g'(x)\)） [11]。
  • >> 警告：切忌在外层求导的同时对内层求导！必须遵循先后步骤。
02 Using Leibniz Notation
Leibniz notation is particularly useful in physics and Related Rates. We introduce an intermediate variable \(u\) to represent the "inside" layer, allowing us to multiply individual rates of change [13]. 莱布尼茨表示法在物理学和相关变化率问题中极为实用。通过引入一个中间变量 \(u\) 来代表“内层”函数，我们可以直接将各自独立的变化率相乘 [13]。
STEP-BY-STEP ALGORITHM [13]
1. Define the inside function as \(u\) (\(u = g(x)\)) [13] 1. 将内层部分定义为中间变量 \(u\) [13]
2. Express outside in terms of \(u\) (\(y = f(u)\)) [13] 2. 用 \(u\) 重新表示外层函数 [13]
3. Calculate \(\frac{dy}{du}\) and \(\frac{du}{dx}\) separately [13] 3. 分别独立求出两个单层导数 [13]
4. Multiply rates: \(\frac{dy}{dx} = \frac{dy}{du} \cdot \frac{du}{dx}\) [13] 4. 将变化率相乘，并将结果中的 \(u\) 替换回 \(x\) 表达 [13]
MACRO-STRUCTURE DETERMINATION [12]
Chain Rule frequently appears combined with Product/Quotient rules. Always identify the macro-structure of the function first to determine which rule is the overarching framework [12]. 链式法则经常与乘积/商法则复合在一起。求导前必须首先判定整道题目的宏观架构，以便决定最外层先套用哪一个框架 [12]。

04 Multi-Layer Onion (Double Chain Rule)
Sometimes functions have 3 or more layers. You simply apply the rule repeatedly: \(f'(g(h(x))) \cdot g'(h(x)) \cdot h'(x)\). Crucial Step: Always rewrite trigonometric functions with exponents to clearly see the outermost power layer [13]! 有时嵌套函数会有 3 层甚至更多层。我们只需连续套用乘法链即可。核心关键：在求导含有高次幂的三角函数时，务必先将幂写在括号最外侧，从而看清最外层的幂结构 [13]！
\[\text{Target Function: } y = \sin^4(5x) \quad \implies \quad \text{Rewrite: } y = [\sin(5x)]^4 \quad [13]\] \[\text{Layer 1 (Outer): } [\text{stuff}]^4 \implies \text{Deriv: } 4[\text{stuff}]^3 \quad [13]\] \[\text{Layer 2 (Middle): } \sin(\text{stuff}) \implies \text{Deriv: } \cos(\text{stuff}) \quad [13]\] \[\text{Layer 3 (Inner): } 5x \implies \text{Deriv: } 5 \quad [13]\]
\[y' = 4[\sin(5x)]^3 \cdot \cos(5x) \cdot 5 \quad \text{[Chain all derivatives together]} \quad [13]\] \[y' = \mathbf{20\sin^3(5x)\cos(5x)} \quad \text{[Simplify]} \quad [13]\]
05 Worked Examples

Example 01: Chain Rule containing a Quotient Rule

Find the derivative of \(y = \left(\frac{2x+1}{x-3}\right)^3\) and simplify algebraically [12]. 求函数 \(y = \left(\frac{2x+1}{x-3}\right)^3\) 的导数并化简 [12]。
SYSTEM LOG: Macro-Structure = Chain Rule, Micro-Structure = Quotient Rule [12]
\[y' = 3\left(\frac{2x+1}{x-3}\right)^2 \cdot \frac{d}{dx}\left[\frac{2x+1}{x-3}\right] \quad \text{[Derived outside, prepped for inside]} \quad [12]\] \[y' = 3\left(\frac{2x+1}{x-3}\right)^2 \cdot \left[\frac{(2)(x-3) - (2x+1)(1)}{(x-3)^2}\right] \quad \text{[Applied Quotient Rule for inside]} \quad [12]\] \[y' = 3\frac{(2x+1)^2}{(x-3)^2} \cdot \left[\frac{2x-6-2x-1}{(x-3)^2}\right] \quad \text{[Simplify numerator]} \quad [12]\] \[y' = \frac{3(2x+1)^2}{(x-3)^2} \cdot \frac{-7}{(x-3)^2} = \mathbf{\frac{-21(2x+1)^2}{(x-3)^4}} \quad \text{[Final algebraic consolidation]} \quad [12]\]

Example 02: Derivative Proof via Chain Rule and Leibniz Notation

Find \(\frac{dy}{dx}\) for \(y = \sqrt{x^2+8x}\) by setting up intermediate variable \(u\) [13]. 通过设置中间变量 \(u\)，计算 \(y = \sqrt{x^2+8x}\) 的导数 \(\frac{dy}{dx}\) [13]。
SYSTEM LOG: Leibniz Rate Multiplication Active [13]
Let \(u = x^2+8x \implies y = u^{1/2}\). 令内层变量 \(u = x^2+8x\)，则外层可化为 \(y = u^{1/2}\)。
\[\frac{dy}{du} = \frac{1}{2}u^{-1/2} = \frac{1}{2\sqrt{u}} \quad \text{and} \quad \frac{du}{dx} = 2x+8 \quad [13]\] \[\frac{dy}{dx} = \frac{dy}{du} \cdot \frac{du}{dx} = \left(\frac{1}{2\sqrt{u}}\right)(2x+8) \quad [13]\] \[\frac{dy}{dx} = \frac{2x+8}{2\sqrt{x^2+8x}} = \mathbf{\frac{x+4}{\sqrt{x^2+8x}}} \quad \text{[Substitute } u \text{ back and simplify]} \quad [13]\]
